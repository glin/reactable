# Print a reactable widget for knitr

This S3 method exists to enable [`reactable()`](reactable.md)'s `static`
rendering option.

## Usage

``` r
knit_print.reactable(x, ...)
```

## Arguments

- x:

  A [`reactable()`](reactable.md) instance.

- ...:

  Additional arguments passed to the S3 method.
